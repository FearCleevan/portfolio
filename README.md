🔗 **[lazandev.vercel.app](https://lazandev.vercel.app/)**

## ✨ Features

### 📱 Modern UI/UX
- **Dark/Light Mode** — System-aware theme with smooth transitions
- **Fully Responsive** — Optimized for 375px to 1440px+ screens
- **Mono/B&W Design** — Clean, professional black-and-white aesthetic
- **Bento Grid Layout** — Modular card-based sections
- **Smooth Animations** — Page transitions, hover effects, loading states
- **Bai Jamjuree Font** — Clean, modern sans-serif typography

### 🧠 AI Chat Assistant
- **Context-Aware** — Answers questions about skills, projects, experience, and certifications
- **Code Snippets** — Sends code examples in JavaScript, Python, SQL, and more
- **Meeting Scheduler** — Inline booking form for consultations
- **Quick Prompts** — Pre-built questions for easy interaction
- **Runs on Vercel AI SDK** — Real-time streaming responses

### 📊 GitHub Contribution Graph
- **7×53 Grid** — GitHub-style calendar showing daily contributions
- **Multiple Years** — Toggle between all available contribution years
- **Real SVG Icons** — Day labels, month labels, and legend
- **GitHub GraphQL API** — Fetches real contribution data

### 📝 Blog System
- **Markdown Support** — Write posts with headings, lists, code blocks, and links
- **Syntax Highlighting** — Prism.js with VS Code-style theme
- **Copy Code Buttons** — One-click code copying
- **Tag Filtering** — Search by title, tag, or content
- **Social Sharing** — LinkedIn, Facebook, and copy-link sharing

### 🏆 Certifications & Education
- **Real Icons** — SVG trophy, star, and award icons
- **Category Badges** — Filterable achievement categories
- **Education Timeline** — Degree, school, period, and location

### 💼 Experience Timeline
- **Vertical Timeline** — Visual career progression
- **Current Role Indicator** — Highlighted with ring effect
- **Detailed Responsibilities** — Bullet-point lists per role
- **Technology Tags** — Stack-per-position display

### 🛠️ Projects Showcase
- **Live Demo Links** — External link icon for deployed projects
- **Tech Stack Tags** — Each project shows its technology stack
- **Image Previews** — Lightbox gallery with keyboard navigation (← → Esc)
- **Category Filtering** — Web, Mobile, AI/ML, Open Source
- **Search** — Filter by title, description, or technology

### 📞 Connect Section
- **Email & Phone** — Direct contact links
- **Calendly Integration** — Schedule meetings directly
- **Social Links** — GitHub, LinkedIn, Facebook, Instagram with real SVG icons

### 🔍 SEO & Performance
- **Open Graph Tags** — Rich previews on social media
- **Twitter Cards** — Large image summaries
- **JSON-LD Schema** — Person and WebSite structured data
- **Sitemap & Robots.txt** — Full XML sitemap with blog posts
- **Google Analytics 4** — Free, privacy-respecting analytics
- **DNS Prefetch** — Optimized external resource loading
- **Custom Scrollbar** — Styled for both light/dark modes

### 🛡️ Security
- **No Secrets Exposed** — All tokens via environment variables
- **GitHub Push Protection** — Repository configured for secret scanning
- **Input Sanitization** — XSS prevention in chat and blog rendering

## 🧰 Tech Stack

### Frontend
- **React 18** — UI library
- **Vite** — Build tool and dev server
- **Tailwind CSS 4** — Utility-first CSS framework
- **React Router 6** — Client-side routing
- **React Icons** — Feather and Simple Icons

### Backend & APIs
- **Vercel AI SDK** — Chat assistant (Gemini/OpenAI)
- **GitHub GraphQL API** — Contribution data
- **Google Analytics 4** — Visitor analytics
- **Calendly API** — Meeting scheduling

### DevOps & Deployment
- **Vercel** — Hosting and serverless functions
- **GitHub Actions** — CI/CD (via Vercel Git integration)
- **Environment Variables** — Secure token management
